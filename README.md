# Criterios de corrección

Se aconseja poner comentarios para su mejor comprensión. Estos comentarios explicarán la funcionalidad del código.

**Importante:**  
Si se detecta la entrega copiada, ya sea de una fuente externa o con un contenido idéntico al de otro alumno/a, serán evaluadas ambas copias con una calificación de 0.

| Descripción | Puntos |
|------------|--------|
| El documento HTML está bien estructurado, utilizando contenedores donde es necesario y siguiendo un contenido semántico correcto. | 1 |
| La estructura general se ha maquetado de manera adecuada, asegurando una disposición responsiva y organizada. | 2 |
| Menú completamente clicables en cualquier área e indicando el elemento actual. | 2 |
| La cabecera contiene correctamente los cuatro elementos (H1, H2, imagen) y están agrupados de manera lógica. La imagen se elige y se presenta adecuadamente. | 1,5 |
| Los enlaces a UAX y la Universidad están correctamente implementados y apuntan a las URLs adecuadas. Los artículos están bien formateados con títulos y párrafos. | 1 |
| La sección de resultados de la comparación muestra "cajas" con el nombre y el valor, maquetadas con Flexbox o Grid. | 2 |
| El pie de página contiene un enlace a la Universidad y el símbolo de copyright se presenta correctamente como “©”. | 0,5 |

**Total: 10 puntos**
